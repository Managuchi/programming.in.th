# Grader

## Stack
React.js, Node.js, Express.js, MongoDB, TypeScript, Babel, Firebase

## How to use

Install for linux:

```bash
sudo apt-get install yarn
```

Install for OSX:

```bash
brew install yarn
```

Add node_modules:

```bash
yarn 
```

To start:

```bash
yarn start
```

To dev:

```bash
yarn dev
```

To deploy:

```bash
firebase deploy
```
