# Grader

## Stack

React.js, Firebase

## Dev setup

1. Create src/config.ts
2. Export firebase config

## How to use

Install for linux:

```bash
sudo apt-get install yarn
```

Install for OSX:

```bash
brew install yarn
```

Add node_modules:

```bash
yarn
```

To start:

```bash
firebase serve
```

To dev:

```bash
yarn dev
```

To deploy:

```bash
firebase deploy
```
